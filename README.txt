
--------- Continut fisiere -------------------------------------

Foldere:
	 *Assets -> In acest folder se gasesc toate fisirele care au contribuit la crearea graficii jocului
		    Ele sunt impartite in subfoldere, pentru fiecare componenta a jocului imparte
	 *Build -> Se gaseste executabilul jocului
	 *GreenBird.sln -> Solutia de visual studio a jocului, in care se gasesc cele 3 script-uri de functionare


----------------- INSTALARE JOC ------------------------------

Jocul nu necesita instalare, pentru a fi rulate se navigheaza in folderul BUILD, unde se ruleaza executabilul GreenBird.exe


-------------------- User Guid --------------------------------

Jocul se joaca prin intermediul mouse-ului. Se apasa pe pasarea verde, se trage pentru a prinde forta, apoi se elibereaza
butonul mouse-ului pentru a lansa pasarea in directia dorita. Dupa eliminarea inamiciilor se trece la urmatorul nivel

PENTRU INCHIDEREA JOCULUI SE APASA ALT+F4
